# 거버넌스 진단 결과

거버넌스 진단 결과를 확인할 수 있습니다. <br />

전체 항목 및 각 리소스 별로 점검 항목들을 확인할 수 있고, 상세 정보 메뉴를 통해 점검 항목의 기본정보, 취약정보, 양호정보, N/A 정보, 상세 조치 방안을 확인할 수 있습니다. 

---
