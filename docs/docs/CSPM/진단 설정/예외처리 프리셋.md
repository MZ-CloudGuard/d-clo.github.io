# 예외처리 프리셋
{: .no_toc }
<br>

## Table of contents
{: .no_toc .text-delta }

1. TOC
{:toc}

진단 결과에 대한 예외처리를 프리셋으로 저장하여 관리할 수 있습니다.<br>
예외처리 프리셋은 상세 진단 결과 화면의 <span style="border: 1px solid #5659F1; border-radius: 2px; padding: 1px 6px;">예외 처리</span> 기능에서 프리셋으로 저장하여 최초 생성할 수 있습니다.<br>
이후 예외처리 수정 또는 삭제가 필요할 경우 예외처리 프리셋에서 작업을 진행할 수 있습니다.<br>

- 예외처리 프리셋이 설정된 대상과 적용 버전 등의 정보를 확인할 수 있습니다.
- 우측의 <span style="color:#4472C4;">진단 시 자동 적용</span> 토글을 활성화하면, 매 진단 시 해당 예외처리 내용이 자동으로 적용되어 진단이 수행됩니다.
- 상세정보 옆 <span style="border: 1px solid #5659F1; border-radius: 1px; padding: 1px 4px;">예외 처리 내용</span> 탭에서는 현재 적용 중인 예외처리 내용을 JSON 형식으로 확인할 수 있습니다.
- 목록에서 원하는 항목을 체크하면 해당 프리셋을 수정하거나 삭제할 수 있습니다.

## 리소스 정보 화면
<center>
    <img
        src="../../../../assets/images/예외처리프리셋_리소스정보.png"

</center>

① 진단 항목(Rule)에 대한 정보를 확인하고, 예외처리를 적용할 항목을 선택할 수 있습니다.

② 진단 결과를 확인하고 적용할 예외처리 결과를 선택할 수 있습니다.
```
 ※ 예시
    - 설정 불가 : 진단 결과 - 취약 / 예외처리 결과 - 취약
    - 설정 가능 : 진단 결과 - 취약 / 예외처리 결과 - 양호 또는 N/A
 ```

③ 예외처리 유형을 선택하고 사유를 작성할 수 있습니다. <br>
- 예외처리 유형<br>
⒜ **진단 항목 예외처리** : 진단 항목(Rule) 전체에 예외처리를 적용합니다. 해당 진단 항목의 모든 리소스에 동일한 예외처리 결과가 적용됩니다.<br>
⒝ **개별 리소스 예외처리** : 선택한 리소스에만 예외처리를 적용합니다.<br>
⒞ **태그 예외처리** : 리소스에 설정된 태그(Key/Value)를 기준으로 예외처리를 적용합니다.<br>

④ 진단 결과(<span style="display:inline-block; padding:1px 3px; border-radius:3px; background-color:#E4E7FF; color:#5459F0; font-weight:600; font-size:0.7em; margin-right:8px;">양호</span>,<span style="display:inline-block; padding:1px 3px; border-radius:3px; background-color:#FFE7D6; color:#F0803C; font-weight:600; font-size:0.7em; margin-right:8px;">취약</span>,<span style="display:inline-block; padding:1px 3px; border-radius:5px; background-color:#E9EAEC; color:#6B7280; font-weight:600; font-size:0.7em;">N/A</span>)에 따른 현재 리소스 정보를 확인할 수 있습니다.

리소스 정보 탭에서 선택한 예외처리 유형의 상세 정보를 확인할 수 있습니다.

- 기존 예외처리된 항목은 상태값이 <span style="display:inline-block; padding:1px 7px; border-radius:9999px; background-color:#9CA3AF; color:#fff; font-weight:600; font-size:0.75em; letter-spacing:0.5px;">APPLIED</span>로 표시되며, 새로 추가한 항목은 <span style="display:inline-block; padding:1px 7px; border-radius:9999px; background-color:#22C55E; color:#fff; font-weight:600; font-size:0.75em; letter-spacing:0.5px;">NEW</span>로 표시됩니다.<br>
- 목록에서 원하는 항목(행)을 클릭하면 예외처리 설정 내용을 JSON 형식으로 확인할 수 있으며,<br>해당 예외처리 항목을 삭제할 수도 있습니다.<br>
- [Reset] 버튼을 클릭하면 기존 및 신규 예외처리 설정이 모두 초기화됩니다.<br>
- [Create] 버튼을 클릭하면 현재 설정으로 새로운 예외처리 프리셋이 생성되며, 버전이 증가합니다.<br>
